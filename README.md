## What is AgencyDash.ai?
White-label AI chatbot dashboard for agencies. Your brand, your clients, your revenue.
- Client portal management under your domain
- AI-powered chatbot configuration
- Automated Stripe billing per client
- Usage analytics and call logs
- Knowledge base management (clients manage their own)

## Who is it for?
- AI chatbot agencies
- Automation consultants
- Digital marketing agencies adding AI services
- Freelancers building AI chatbots for clients

## Tech Stack
- **Frontend:** Next.js 14 + TypeScript + TailwindCSS
- **AI:** Claude claude-opus-4-5 for bot config generation
- **Payments:** Stripe (Starter $49, Growth $99, Scale $249/mo)
- **Deploy:** Vercel

## Quick Start
```bash
git clone https://github.com/Jaiprathap26/agencydash-ai.git
cd agencydash-ai
npm install
cp .env.example .env.local
npm run dev
```
